Website #1 : Online Diff Tool


https://www.diffnow.com


A quick, online diff tool when you want to compare a couple of clipboard snippets.

(There are many of these, but this is the one I like best.)

Pluses: Nice readable output. Step through changed blocks.

Minuses: A little slow to process requests. Occasionally broken/not online.

Website #2 : JSON Tree Viewer and Clean Upper


http://www.jsoneditoronline.org/


Get a handle on big, messy JSON blobs.

(Again this is my favorite of several. So well done!)

Use the tree to dig through really involved structures, or transfer back/forth to pretty print a modest mess.

Website #3 : Interactive CSS/jQuery Selector Testing


http://sk.kapsi.fi/interactive-jquery-tester.html


Interactively see what part of a document a CSS/jQuery selector is actually highlighting.

(Oh, how I wish I knew about this back when I started working with selectors.)

Note that the element tree does NOT show the root element. May need to wrap your snippet.
